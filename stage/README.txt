Vehicle Control
===============
A small script imitating the GTA:O remote vehicle control interaction menu.

Requirements
============
Grand Theft Auto V
ScriptHookV

Installation
============
Put VehicleControl.asi and the VehicleControl folder in the main GTA V folder.

Usage
=====
Press F6 to show the menu, or 
Enter the "vehiclecontrol" cheat, or
Press RB + RStick

Changing hotkey
===============
Check settings_menu.ini.

* Available keys in Keyboard_Keys.txt
* Available controller buttons in Controller_Buttons.txt
* CheatString can be anything you want
